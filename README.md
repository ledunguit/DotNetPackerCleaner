# DotNetPackerCleaner
Cleaner/Deobfuscator for DotNetPacker/DotNetPatcher (tested on v4.6.6.0)

# Is able to
<pre>
- Decrypt Strings (they're still in their functions, but decrypted)
- Disables Protections (AntiTampering, AntiDumping, AntiDebugging)
- Removes AntiDe4dots
</pre>
